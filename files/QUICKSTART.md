## The Fastest Way to Get Online: GitHub Pages

### Step 1: Create GitHub Account (2 minutes)
1. Go to: https://github.com
2. Click "Sign up"
3. Enter your email, create password
4. Verify your email

### Step 2: Create Repository (1 minute)
1. Click the "+" in top right â†’ "New repository"
2. Repository name: `danibrands-website`
3. Select "Public"
4. Click "Create repository"

### Step 3: Upload Your Files (1 minute)
1. Click "uploading an existing file"
2. Drag ALL your DaniBrands files into the upload area
3. Click "Commit changes"

### Step 4: Enable GitHub Pages (1 minute)
1. Click "Settings" tab
2. Scroll to "Pages" in left sidebar
3. Under "Source", select "main" branch
4. Click "Save"
5. Wait 1-2 minutes
